PENJELASAN HUKUM OHM - RESISTOR SERI (Rausyan Jaisyu Muhammad)

 Kode program C ini merupakan kalkulator listrik sederhana yang digunakan untuk menghitung besaran listrik dasar. Program terdiri dari beberapa fungsi, yaitu `kalkulatorOhm()` untuk menghitung tegangan, arus, atau hambatan berdasarkan Hukum Ohm, `kalkulatorDaya()` untuk menghitung daya listrik menggunakan rumus P = V Ã— I, serta `resistorSeri()` untuk menghitung total hambatan pada rangkaian resistor seri dengan cara menjumlahkan nilai setiap resistor. Setiap fungsi menerima input dari pengguna, memproses data sesuai rumus yang digunakan, dan menampilkan hasil perhitungan, serta menggunakan `getchar()` untuk menahan tampilan agar hasil dapat dilihat sebelum program berlanjut.
